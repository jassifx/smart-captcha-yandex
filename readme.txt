Интеграция Yandex Smart Captcha для вашего WordPress сайта.

== Description ==
Yandex Smart Captcha интегрируется в формы:
- отправки комментария
- авторизации
- регистрации
- восстановления пароля

**ВНИМАНИЕ!** Сервис Yandex Smart Captcha в данный момент находится на стадии Preview, поэтому нужно [подавать заявку](https://cloud.yandex.ru/services/smartcaptcha) на использование.
Также отбратите внимание на [тарифы Яндекса](https://cloud.yandex.ru/docs/smartcaptcha/pricing) - бесплатно до 250 тысяч запросов в месяц.

[Инструкция](https://cloud.yandex.ru/docs/smartcaptcha/quickstart) как получить API ключи Yandex Smart Captcha.
После получения ключей, введите их в настройках плагина и сохраните настройки. Всё, плагин работает.

Больше функций в будущих обновлениях.

== Frequently Asked Questions ==

= Почему капча не отображается в других формах на сайте? =
Потому что на данный момент ещё не готова интеграция с данным плагином форм или вы используете свою кастомную форму.
